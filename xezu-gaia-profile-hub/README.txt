# The Xezu Hub - Gaia Style Profile Prototype

## How to open it
1. Extract the zip file.
2. Open `index.html` in your browser.

## What this version includes
- Gaia-style profile layout
- Profile banner
- Avatar card
- Badges
- Game tracker
- Journal section
- Comment wall prototype
- Support links

## Important
The comment wall is only a front-end prototype.
Comments do not save after refreshing yet.
